# 🐞 Bug Tracker System (Python)

A simple command-line based Bug Tracking System built using Python. This project simulates a basic Software Quality Assurance (QA) workflow where bugs can be reported, tracked, and updated.

---

## 🚀 Features
- Add new bug reports  
- Assign priority levels (Low / Medium / High)  
- View all reported bugs  
- Update bug status (Open → Closed)  

---

## 🛠️ Technologies Used
- Python (Core Programming)
- Data Structures (Lists, Dictionaries)

---

## 💡 Concepts Applied
- Structured Programming  
- Data Management  
- Basic QA Workflow Simulation  
- Problem-Solving & Logic Building  

---
